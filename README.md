# UX Research Agent Dashboard

Next.js와 Tailwind CSS로 만든 UX 리서치 에이전트 대시보드입니다.

## 구조

- **왼쪽 사이드바**: 리서치 히스토리 (세션 목록, 검색, 새 리서치 시작)
- **오른쪽 영역**: 채팅 인터페이스 (에이전트와 대화)

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 으로 접속하세요.

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
